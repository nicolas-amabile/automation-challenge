# Automation challenge
Google Calculator

## Instructions
### Install
```
> git clone git@github.com:nicolas-amabile/automation-challenge.git
> cd automation-challenge
> npm install
# Add the baseUrl to the .env file
> echo "CYPRESS_BASE_URL='https://www.google.com'" >> .env
```

### Run
#### Open Cypress
```
npm run cy:open
```

#### Headless mode
```
npm run cy:run
```
